# Privacy Notes

This repository is designed for public viewing.

## Included

- Public-safe career summary.
- Generic automation architecture.
- Fictional demo examples.
- High-level tools and skills.
- Links to public GitHub and LinkedIn profiles.

## Excluded

- Employer source code.
- Private repository structure.
- Production or staging URLs.
- API keys, tokens, passwords and secrets.
- Customer, booking, financial or personal data.
- Internal screenshots, reports or logs.
- Current-employer implementation details.

## Rule

When in doubt, describe the skill and remove the private system.
